Vicor BCM™High Power Series - Technical Details & Specifications

Modular Power / BCM • January 1, 2025 • Technical Article

The BCM High Power Series features high-efficiency current multiplication bus converters with bidirectional power flow designed for high-current applications. These modules utilize Vicor's advanced technology to multiply current while reducing voltage, preserving power delivery dynamics, and making them ideal for applications requiring high current at lower voltage from a higher voltage source.

Overview

The BCM High Power Series represents Vicor's solution for high-current applications requiring current multiplication with preserved dynamics. These devices are specifically optimized for high-power systems where current multiplication from a higher voltage source is needed.

The fixed-ratio transformation (typically 1:4, 1:2, or 1:1) makes them ideal for applications where you need high current at a lower voltage from a higher voltage source. The bidirectional design enables power flow in both directions, and the non-isolated design provides optimal power density.

Key Features

Current Multiplication

BCM modules multiply current while reducing voltage based on the turns ratio, ideal for high-current, low-voltage applications.

Bidirectional Power Flow

Supports power flow in both directions, enabling applications like battery systems and regenerative power conversion.

Non-Isolated Design

Efficient non-isolated design maximizes power density while preserving power delivery dynamics to the load.

High Power Density

Delivers high current multiplication in compact VIA package, providing excellent power density for space-constrained applications.

Applications

BCM High Power modules are optimized for high-current applications requiring current multiplication with preserved dynamics. Common applications include:

High-Current Computing

Powering high-current processors and GPUs where high current at low voltage is needed from a higher voltage bus.

Motor Drives

High-current motor control applications requiring bidirectional power flow and current multiplication.

Battery Systems

Battery pack conversion and charge systems with bidirectional power flow capabilities.

Design Considerations

Power Delivery Network

BCM modules preserve the power delivery network dynamics to the load, meaning that the load "sees" the upstream source characteristics. This can be advantageous in applications where load transient requirements are met by the upstream source.

Layout Recommendations

For optimal performance in high-current applications, ensure adequate copper for current carrying and proper thermal management. Minimize current loops to reduce inductance and electromagnetic interference. Pay attention to input and output capacitor placement to support the high-frequency switching characteristics.

System Integration

BCM High Power modules excel in applications where you need high current multiplication with preserved dynamics. They are ideal for applications where load transients are handled by the upstream power source.

Frequently Asked Questions

How does current multiplication work in BCM modules?

BCM modules use a fixed turns ratio transformer to multiply current while reducing voltage. For example, with a 1:4 ratio, the output current is 4x the input current at 1/4 the voltage, maintaining power (minus losses).

What is the difference between BCM and DCM modules?

BCM modules provide current multiplication with bidirectional power flow in a non-isolated design. DCM modules provide voltage transformation with galvanic isolation but no current multiplication.

Can BCM modules handle bidirectional power flow?

Yes, BCM modules are bidirectional by design, allowing power to flow from input to output or output to input, making them ideal for applications like battery charging/discharging.

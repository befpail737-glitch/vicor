The BCM Low Profile Series features compact, high-efficiency current multiplication bus converters with bidirectional power flow designed for space-constrained applications. These modules utilize Vicor's advanced LVM (Low Vertical Mount) packaging technology to deliver current multiplication in a low-profile form factor, making them ideal for applications requiring high current at low voltage where height is limited.

Overview

The BCM Low Profile Series represents Vicor's solution for space-constrained applications requiring current multiplication with preserved dynamics. These devices are specifically optimized for applications where vertical space is limited but current multiplication from a higher voltage source is needed.

The fixed-ratio transformation (typically 1:4, 1:2, or 1:1) makes them ideal for applications where you need high current at a lower voltage from a higher voltage source. The bidirectional design enables power flow in both directions, and the non-isolated design provides optimal power density in a low-profile package.

Key Features

Current Multiplication

BCM modules multiply current while reducing voltage based on the turns ratio, ideal for high-current, low-voltage applications in tight spaces.

Low Profile Design

LVM packaging technology provides a compact, low-profile solution ideal for space-constrained applications with height restrictions.

Bidirectional Power Flow

Supports power flow in both directions, enabling applications like battery systems and regenerative power conversion in compact form.

High Efficiency

Achieves up to 97.5% efficiency while in a low-profile package, minimizing thermal management requirements in tight spaces.

Design Considerations

Power Delivery Network

BCM modules preserve the power delivery network dynamics to the load, meaning that the load "sees" the upstream source characteristics. This can be advantageous in applications where load transient requirements are met by the upstream source, even in compact spaces.

Layout Recommendations

For optimal performance in space-constrained applications, ensure adequate copper for current carrying and proper thermal management within the limited vertical space. Minimize current loops to reduce inductance and electromagnetic interference in tight layouts.

System Integration

BCM Low Profile modules excel in applications where you need high current multiplication with preserved dynamics in a compact form factor. They are ideal for applications where load transients are handled by the upstream power source and space is at a premium.

Frequently Asked Questions

How does current multiplication work in BCM modules?

BCM modules use a fixed turns ratio transformer to multiply current while reducing voltage. For example, with a 1:4 ratio, the output current is 4x the input current at 1/4 the voltage, maintaining power (minus losses).

What is the difference between BCM and DCM modules?

BCM modules provide current multiplication with bidirectional power flow in a non-isolated design. DCM modules provide voltage transformation with galvanic isolation but no current multiplication.

Can BCM Low Profile modules handle bidirectional power flow?

Yes, BCM modules are bidirectional by design, allowing power to flow from input to output or output to input, making them ideal for applications like battery charging/discharging in compact systems.

The PRM Low Profile Series features compact, high-efficiency regulated bus converters designed for Factorized Power Architecture (FPA) systems in space-constrained applications. These modules provide regulated output voltage when used with Vicor VTMs (Vicor Transformer Modules) to create a complete power solution with exceptional efficiency, power density, and dynamic performance in a low-profile LVM package.

Overview

The PRM Low Profile Series represents Vicor's solution for space-constrained applications requiring regulated power conversion in Factorized Power Architecture (FPA) systems. PRM (Primary Regulated Module) modules work in conjunction with VTMs (Vicor Transformer Modules) to provide a two-stage power conversion system. The PRM provides regulation and power factor correction, while the VTM provides current multiplication and isolation.

This architecture separates regulation from current multiplication, allowing each stage to be optimized independently for maximum efficiency and performance. The LVM package provides the solution in a low-profile form factor ideal for applications with height restrictions.

High Efficiency

Design Considerations

Factorized Power Architecture

PRM modules must be used in conjunction with VTMs to form a complete power solution. The PRM provides regulation and power factor correction, while the VTM provides current multiplication and isolation. The output of the PRM connects to the input of the VTM for the complete FPA solution.

Thermal Management

In space-constrained applications, thermal management becomes critical. Consider airflow and thermal vias to PCB copper to manage heat dissipation in tight spaces. The low-profile design may require additional thermal management strategies compared to standard packages.

System Integration

PRM Low Profile modules excel in applications where you need regulated power with high efficiency in a compact form factor. They are ideal for systems where the load requirements match VTM output capabilities when combined with PRM regulation.

Frequently Asked Questions

What is Factorized Power Architecture (FPA)?

FPA is Vicor's approach to power system design that separates regulation from current multiplication using PRM and VTM modules. This allows optimization of each function independently, resulting in higher efficiency, power density, and performance.

How do PRM and VTM modules work together?

The PRM provides regulation and power factor correction, while the VTM provides current multiplication and isolation. The PRM output connects to the VTM input to create a complete regulated, isolated power solution.

What are the benefits of LVM packaging?

LVM packaging provides a low-profile solution ideal for applications with height restrictions while maintaining high efficiency and power density.
